# Meteoros Floripa - Site

![Build](https://github.com/Meteoros-Floripa/site/workflows/Build/badge.svg)

Captures of [BRAMON](https://www.bramonmeteor.org) from [TLP](https://www.mrprompt.com.br) meteor stations.

## Running local

```console
docker-compose up
```
